Ultra Sparse Matrix Factorization for Recommender Systems

This project explores the limits of Biased Latent Matrix Factorization (BLMF) under extreme sparsity conditions using the MovieLens 32M dataset. The objective was to identify the minimum rating density required for building a robust recommender system suitable for early-stage companies with limited data. We implemented a Baseline Regularized Bias Model and compared it against BLMF across 10 sparsity levels, testing over 600 model permutations. The BLMF model leveraged Stochastic Gradient Descent (SGD) and Mini-Batch SGD for optimization, with hyperparameter tuning for latent dimensions (K), learning rate, and regularization. Evaluation metrics included RMSE, MAE, Precision@K, and Recall@K, consolidated into a synthesized score for performance comparison. Results showed BLMF consistently outperformed the baseline except at the most extreme sparsity, demonstrating its resilience for personalized recommendations in data-constrained environments. Key Technologies & Tools: Python (NumPy, Pandas, Matplotlib, Scikit-learn) Matrix Factorization Algorithms (FunkSVD-based BLMF) Optimization: Stochastic Gradient Descent (SGD), Mini-Batch SGD Dataset: MovieLens 32M (subset of 150,000 ratings) Evaluation: RMSE, MAE, Precision@K, Recall@K
